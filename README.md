# 📄 Form Site - Premium Response Manager

A high-end, responsive Django application for collecting and managing customer feedback with an elite glassmorphic UI.

---

## 🚀 Key Features

* **✨ Premium Glassmorphism:** Ultra-modern translucent UI with backdrop-filter effects.
* **📱 Universal Response:** Mobile-first architecture for seamless cross-device usage.
* **📊 Dynamic Dashboard:** Real-time feedback monitoring with a professional management table.
* **🛡️ Django Powered:** Secure, scalable, and robust backend implementation.
* **🎨 Micro-animations:** Smooth cubic-bezier transitions and interactive hover states.

---

## 🛠️ Tech Stack

* **Backend:** Python 3.x, Django Framework
* **Frontend:** HTML5, Modern JS (ES6+), Tailwind CSS
* **UI/UX:** Glassmorphism, Lucide Icons, Google Fonts (Poppins)

---

## ⚙️ Quick Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/DasturchiMadaminjon/form_site.git](https://github.com/DasturchiMadaminjon/form_site.git)
   cd form_site
Setup Virtual Environment:

Bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install django
Database Setup:

Bash
python manage.py migrate
Launch Server:

Bash
python manage.py runserver
